# COUP RDC

**COUP RDC** is a mobile-first health training platform for francophone Africa — built for NGOs and health programs (e.g. vaccination campaigns, community health worker training).

- **UI language:** French (`fr-FR`)
- **Stack:** React + Vite + Tailwind CSS + Convex
- **Roles:** Learner, Lead, Admin, Super Admin

## Quick start

```bash
npm install
# Add VITE_CONVEX_URL to .env.local
npm run dev
```

Demo login: `learner@helty.africa` / `demo1234` (see [SETUP.md](./SETUP.md) for all accounts).
